
# Installation

The package is published on [PyPI](https://pypi.org/project/clash-royale-python/) and can be installed with `pip` (or any equivalent):

::::{tab-set}

:::{tab-item} uv
uv add clash-royale-python
:::

:::{tab-item} poetry 
poetry add clash-royale-python
:::

:::{tab-item} pip 
pip install clash-royale-python
:::

::::
